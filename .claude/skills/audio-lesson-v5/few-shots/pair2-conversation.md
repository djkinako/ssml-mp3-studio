# Few-shot ペア2: 帶把傘出門吧(会話文 + 語気助詞)

## ペア概要

| 項目 | 内容 |
|---|---|
| 日本語 | 今日は雨が降りそうだから傘を持っていこう |
| 中国語(繁体字) | 今天好像會下雨,帶把傘出門吧 |
| カテゴリ | 会話文 + 語気助詞「吧」(機能語トラック分離例) |
| 文字数 | 14字(短文 → Phase 4 は 2 ポイント) |
| 由来 | v4 入力2(日常会話、代名詞なし)を v5-modular 4ブロック化 |
| 適用軸(v1.5.1) | C: 語気助詞(吧) / D: 量詞特殊用法(把) |

## Phase 0 フォーカス語(5列・v1.5.1)

| # | 単語 | ピンイン(カタカナ) | 日本語訳 | 選定理由 |
|---|---|---|---|---|
| 1 | 今天 | ジンティエン | 今日 | 時間語、文の意味のコア、口語の定番 |
| 2 | 好像會 | ハオシャンホイ | 〜しそう | 推量マーカーの定番複合、N1-N2 学習者がモヤるポイント |
| 3 | 帶把傘 | ダイバーサン | 傘を持っていく | 量詞「把」の特殊用法、把字構文ではない型の例 |
| 4 | 出門 | チューメン | 外出する | 動補構造の口語動詞、短文補完で4枠目を埋める標準枠 |

**自己相互チェック3点**:
- ① 部分文字列共有なし: 今天/好像會/帶把傘/出門、互いに包含関係なし ✅
- ② 同一品詞・意味カテゴリの重複なし: 時間名詞・助動詞句・動詞句・動詞でバラけている ✅
- ③ Phase 3 派生語列挙が交差しない: 今天→明天/昨天系、好像→似乎/大概系、帶→帶走/帶來系、出門→出去/出國系 ✅

## 機能語特例トラック(0枠目・v1.5.1)

Q6 ルール(機能語は Phase 0 4枠対象外)に従い、語気助詞「吧」は通常の4枠表から除外して、別表「機能語特例トラック」で扱う。

| 機能語 | ピンイン(カタカナ) | 役割 | 扱い方 |
|---|---|---|---|
| 吧 | バ | 語気助詞(文末、提案・推量・同意) | Phase 0 4枠外、Phase 2 で言及のみ、Phase 3 深掘り省略、**Phase 4 機能語トラックで扱う** |

→ これが「機能語特例トラック分離」の実例。Phase 4 の文末詞ポイントで詳しく扱う。

## 学習ポイント

- **会話文の温度感**: 書面語スイッチが入らない、日常の口語表現
- **機能語の扱い**: 「吧」を Phase 0 4枠から除外し、Phase 4 で文末詞軸で扱う模範例(Q6 適用例)
- **Phase 4 「2ポイント」動的化**: 短文(14字)で文法的に淡白なので、3ポイント固定ではなく **2ポイントで止める** 模範例
- **量詞「把」の特殊用法**: 把字構文ではなく「一把傘」の量詞残り
- **Phase 2 軽量版(v1.5.0)**: 3パターン判定テンプレ(①日本語と同じ字 / ②旧字体 / ③日本語にない繁体字)のみで構成、構成解説・印象表現・字形描写は一切排除
